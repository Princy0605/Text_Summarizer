# 📝 Text Summarizer using Fine-Tuned Hugging Face Transformer Model

## 🚀 Overview

Text Summarizer is a Generative AI project that automatically generates concise and meaningful summaries from long text inputs. The project leverages a fine-tuned Transformer model from Hugging Face to produce high-quality summaries while preserving the core information of the original content.

The application provides an easy-to-use web interface built with HTML, CSS and JS, while FastAPI serves as the backend API for model inference and deployment.

## ⚙️ Model Training

The model was fine-tuned on a dialogue summarization dataset using the Hugging Face Transformers library.

### Training Pipeline

* Data Cleaning and Preprocessing
* Tokenization using T5 Tokenizer
* Fine-Tuning T5 Transformer Model
* Model Evaluation
* Saving Trained Model and Tokenizer
* Deployment using FastAPI

---

## 📊 Deep Learning Workflow

1. Collect and preprocess text data
2. Clean and normalize text
3. Tokenize input text
4. Fine-tune T5 Transformer model
5. Save trained model
6. Build FastAPI backend
7. Create frontend using HTML, CSS and JS
